# Credentials
credentials go in a json file named `cred.json` in this folder

```json
{
    "client_id":"<client_id>",
    "client_secret":"<client_secret>"
}
```

# Tokens
token info is stored here in a file named `tokens.json`

```json
{
    "access_token":"<access_token>",
    "refresh_token":"<refresh_token>"
}
```
